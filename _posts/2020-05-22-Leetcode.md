## 26. 删除排序数组中的重复项
双指针，如果重复就和前面相邻的互换。
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n, i = len(nums), 0
        if not n:
            return n
        for j in range(1,n):
            if nums[i] != nums[j]:
                i = i + 1
                nums[i] = nums[j]
        return i + 1   
```

## 973. 最接近原点的 K 个点
堆算法，Python一行。
```python
class Solution:
    def kClosest(self, points: List[List[int]], K: int) -> List[List[int]]:
        return heapq.nsmallest(K, points, key=lambda x: sqrt(x[0]*x[0] + x[1]*x[1]))
```
